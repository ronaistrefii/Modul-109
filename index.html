Arbeitsschritte: GitHub Repository erstellen und nutzen

1. Repository auf GitHub erstellen

Ich habe auf GitHub ein neues Repository mit dem Namen Modul-109 erstellt.

2. Repository in der Linux-VM klonen

In der VM habe ich folgendes Kommando ausgeführt, um das Repository zu klonen:

git clone https://github.com/ronaistrefii/Modul-109.git

3. In das geklonte Verzeichnis wechseln

Mit folgendem Befehl habe ich in das Verzeichnis Modul-109 gewechselt:

cd Modul-109

4. README-Datei erstellen

Mit dem Befehl touch README habe ich eine neue Datei namens README erstellt.

5. Änderungen committen

Ich habe meine Änderungen wie folgt committet:

git commit -m "Neues Repository für Modul 109"

6. Änderungen pushen

Ich habe mein Repository mit folgendem Befehl auf GitHub gepusht:

git push -u origin main

  • Username: ronaistrefii
